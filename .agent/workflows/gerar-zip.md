description: Como gerar o pacote ZIP do plugin para instalação no WordPress
---

Sempre que concluir uma tarefa ou quando solicitado pelo usuário, siga estes passos para gerar o pacote ZIP:

1. Garanta que todas as alterações foram salvas.
2. Execute o script de geração:
   ```bash
   bash /home/thiago/Projetos/ser/SER-metodologia-mt24/generate-zip.sh
   ```
3. O arquivo final estará disponível em:
   `/home/thiago/Projetos/ser/SER-metodologia-mt24/metodologia-leitor-apreciador/metodologia-leitor-apreciador.zip`
4. Notifique o usuário que o pacote foi gerado.
